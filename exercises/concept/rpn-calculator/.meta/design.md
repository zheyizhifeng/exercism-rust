# Design

## Goal

Introduce the design idiom of using `Vec` as a stack.

## Learning objectives

- Know what `push()` and `pop()` do.
- Know how to use the values returned by `pop()`.
- Know how to create and use a stack.

## Out of scope

- Mathematical edge cases, like division by zero

## Prerequisites

- intro-fn
- numbers
- option
- enums
- match-basics
- loops
- mutability
